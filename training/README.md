# Training Materials

This directory contains all training materials for the Model Deployment and Monitoring tutorial.

### 1. TRAINING_OUTLINE.md
Complete training session outline with:
- Session structure
- Learning objectives
- Hands-on exercises
- Time allocation
- Assessment questions

### 2. SLIDES_CONTENT.md
Content for presentation slides including:
- 30+ slides covering all topics
- Code examples
- Diagrams and architecture
- Best practices

### 3. AWS_SETUP_GUIDE.md

## Using These Materials

### For Instructors

1. **Review the outline** (`TRAINING_OUTLINE.md`)
2. **Create slides** from `SLIDES_CONTENT.md`
3. **Prepare AWS accounts** using `AWS_SETUP_GUIDE.md`
4. **Set up demo environment** before the session
5. **Prepare exercises** for hands-on practice

### For Students

1. **Read the outline** to understand what you'll learn
2. **Set up AWS account** following `AWS_SETUP_GUIDE.md`
3. **Follow along** with the slides
4. **Complete exercises** during the session
5. **Review materials** after the session

## Session Structure

The training is designed for a **4-6 hour session** with:

- **Theory**: 40% (slides and explanations)
- **Hands-on**: 50% (exercises and demos)
- **Q&A**: 10% (discussion and troubleshooting)

## Prerequisites

Before the session, students should have:

- Basic Python knowledge
- Understanding of REST APIs
- Basic Docker knowledge (helpful)
- AWS account (we'll help set up)
